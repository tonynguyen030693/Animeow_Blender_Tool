================================================
		INSTALLATION
================================================

1. Place "MC_Bakery.py" in your scripts folder.
   It should be something like this:

   Win XP: "C:\Documents and Settings\USERNAME\My Documents\maya\scripts"
   Win Vista/7/8: "C:\Users\USERNAME\Documents\maya\scripts"
   Mac: "/Users/USERNAME/Library/Preferences/Autodesk/maya/scripts"


2. Place "mc" folder in the same folder:

   Win XP: "C:\Documents and Settings\USERNAME\My Documents\maya\scripts"
   Win Vista/7/8: "C:\Users\USERNAME\Documents\maya\scripts"
   Mac: "/Users/USERNAME/Library/Preferences/Autodesk/maya/scripts"

   Alternatively you can place it in icons folder:

   Win XP: "C:\Documents and Settings\USERNAME\My Documents\maya\2014\prefs\icons"
   Win Vista/7/8: "C:\Users\USERNAME\Documents\maya\2014\prefs\icons"
   Mac: "/Users/USERNAME/Library/Preferences/Autodesk/maya/2014/prefs/icons"


3. In Maya open the Script Editor and go to the Python tab, copy and paste the code in MC_Bakery.py or source the script directly.
