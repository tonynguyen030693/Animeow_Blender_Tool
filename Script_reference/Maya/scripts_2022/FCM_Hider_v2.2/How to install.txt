Installation:

1) Put the folder "Icons_Hider" in this location:
C:\Users\YourUser\Documents\maya\201x\prefs\icons

2) Drag and drop the file "Drag_and_drop_Install_FCM_Hider" into the viewport. 
It will create a new shelf button.

Done!

If the drag and drop doesn't work you can create the shelf button manually by copying the content
of "FCM_Hider Shelf button.txt" and pasting it into a shelf button, keep in mind to change "language" to python in the shelf window
