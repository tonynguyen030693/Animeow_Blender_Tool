# Take Note - 2026-07-08

Hôm nay đã hoàn thành việc tích hợp Studio Library API vào quy trình Publish File để tự động xuất dữ liệu chuyển động (Animation Export) chuẩn bị cho bước Gộp Cảnh (Merge/Combine):

## 1. Bản đồ thư mục Studio Library tối ưu
* Thư mục thư viện chính thức cho từng tập phim:
  `Z:/Animeow_Production/{Project}/{Episode}/Published/StudioLibrary/`
* Thư mục phân chia theo từng Shot:
  `.../Published/StudioLibrary/Shot_{shot_num}/`

## 2. Tích hợp Tự động Xuất Anim khi Publish
* Khi artist click chuột phải chọn **🚀 Publish File này (Copy nhanh)**:
  * Tool sẽ đối chiếu xem tệp được chọn có phải là scene đang mở trong Maya hay không.
  * Nếu khớp (đang làm trên scene hiện tại), tool thực hiện copy file Maya sạch lên server và chạy ngầm quy trình xuất chuyển động qua Studio Library API (`mutils.saveAnim`).
  * Nếu không khớp (publish offline tệp khác), tool hiển thị thông báo hướng dẫn: *"Bạn đang publish offline một tệp không phải scene đang mở hiện tại, vì vậy hệ thống không thể xuất dữ liệu anim..."*.

## 3. Cấu trúc xuất Anim linh hoạt
Khi xuất anim, tool tự động chia dữ liệu thành 2 phần:
* **all_assets.anim:** Lưu chuyển động của tất cả Rig có keyframe trong scene cùng lúc, phục vụ trực tiếp cho khâu Gộp Cảnh tự động 1-click.
* **{Namespace}.anim:** Các file anim tách lẻ theo namespace của từng Rig (nhân vật/vật thể) để artist có thể kéo thả sử dụng linh hoạt qua UI Studio Library khi cần.

## 4. Viết mới Animeow Maya Toolboard (Python 3) cho Maya 2022+
* Đã chuyển đổi và viết mới toàn bộ gói `animeow_maya_toolboard` đặt trong thư mục `Maya_script_2022_python_3`.
* Loại bỏ tính năng Ghép nối cũ (Anim Combiner).
* Tích hợp tính năng **Smart Link (Double Locator System)** hoạt động tương tự Blender:
  * **Link (Locator Pair):** Tạo cặp locator `parent` (Hook) và `child` (Offset) lồng nhau, tự động ghi chuyển động cũ thế giới để nạp sang locator con.
  * **Switch Target:** Chuyển đổi Driver dẫn đường sang đối tượng khác tại frame chỉ định mà không giật lệch vị trí trực quan.
  * **Bake & Clean:** Nướng lại chuyển động thực tế cho vật bị dẫn, gỡ constraint và tự động dọn sạch locator dư thừa.
  * **Smart Clean (Tùy chỉnh):** Giữ nguyên lưới Grid Step (ví dụ step=2: 1, 3, 5, 7...) kết hợp giữ nguyên chính xác các keyframe nguồn của Locator/Target nằm ngoài lưới (ví dụ: key cực trị ở frame 4) để artist dễ quản lý key mà không mất chuyển động chính.
* Tích hợp Menu, Shelf Button (thông qua `install.mel` kéo thả) tiện lợi cho artist.
* **Tích hợp 4 Công cụ Tiện ích mới:** Thêm GroupBox khởi chạy nhanh cho **Studio Library, DWPicker, Tween Machine và aTools** trên giao diện Toolboard. Hệ thống sẽ tự động cấu hình nạp các đường dẫn Python & MEL cần thiết từ thư mục tham chiếu `scripts_2022` của dự án.
