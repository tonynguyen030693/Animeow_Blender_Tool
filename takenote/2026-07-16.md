# Nhật ký công việc - 16/07/2026

Hôm nay chúng ta đã hoàn thiện tính năng **Selection Sets Manager** trong bộ công cụ **Animeow Maya Toolboard** (Python 3, Maya 2022+) với các nâng cấp lớn:

## 1. Bộ lọc Set Hệ Thống (Bug Fix)
- Bỏ qua các set ẩn mặt tự động sinh ra bởi Maya như `defaultHideFaceDataSet` và `*HiddenFacesSet` khỏi danh sách quản lý.

## 2. Tổ chức Gom Nhóm Tự Động (`Animeow_sets`)
- Mọi Selection Set khi tạo mới qua tool sẽ tự động gom nhóm dưới dạng con của set cha tên là `Animeow_sets`.
- Ẩn node cha `Animeow_sets` này khỏi UI quản lý của tool để giữ giao diện sạch sẽ.
- Thêm menu chuột phải **"Đưa vào group Animeow_sets"** để gom nhóm nhanh các set cũ.

## 3. Giao diện Cây Phân Cấp (Hierarchical Tree View) & Subsets
- Chuyển đổi bảng danh sách phẳng (Table) thành dạng cây **QTreeWidget** để quản lý quan hệ cha-con (subset/nested sets) trực quan.
- **Tạo set con**: Nếu chọn 1 set trong cây và bấm tạo mới, set đó sẽ tự động trở thành subset của set đang chọn.
- **Sắp xếp lại**: Thêm tuỳ chọn chuột phải **"Nhóm vào Set khác..."** và **"Đưa ra group gốc Animeow_sets"** để kéo thả/nhóm các set nhanh.
- Điều chỉnh cột Tên Set sang cột 0 (chứa các nút thụt dòng cây) và biểu tượng mắt sang cột 1 giúp các icon mắt luôn thẳng hàng và không bị Qt cắt mất.

## 4. Ẩn/Hiện Đệ Quy (Recursive Visibility)
- Khi ẩn/hiện group cha, tool sẽ tự động quét đệ quy và ẩn/hiện toàn bộ các thành viên thuộc các group con (subset) bên trong nó, đồng thời đóng/mở toàn bộ biểu tượng mắt tương ứng.

## 5. Xuất/Nhập File (Export/Import) & Map Namespace
- **Xuất File (Export)**: Xuất các set được chọn ra file `.json` (tự động đệ quy gom hết các set con bên trong set được chọn để tránh mất dữ liệu).
- **Nhập File (Import)**: Đọc file `.json` và nhập lại sets vào scene.
- **Smart Namespace**: Tự động map tên sets và các đối tượng bên trong theo Namespace của vật thể đang chọn trong Viewport (rất hữu ích khi import sets của rig reference).
- **2-Phase Import**: Sửa lỗi mất quan hệ cha-con khi import bằng cơ chế 2 bước: tạo toàn bộ set trống trước, sau đó mới điền thành viên và kết nối cha-con.

## 6. Tiện ích Vùng chọn Component
- Bổ sung nút **Grow (>)** (mở rộng chọn mặt/đỉnh/cạnh).
- Bổ sung nút **Shrink (<)** (thu nhỏ chọn mặt/đỉnh/cạnh).
- Bổ sung nút **Toggle Mode (F8)** (chuyển đổi qua lại giữa chế độ chọn Object và Component).

## 7. Cập nhật Shelf Animeow
- Tạo icon trong suốt Cyan neon chuyên nghiệp `sets_icon.png` và lưu vào thư mục `icons/`.
- Tích hợp nút chạy độc lập `Sets` vào file định nghĩa shelf `core/shelf.py` sử dụng icon mới.
