### 2. Su co "Loi ao" SyntaxError & Bai hoc kinh nghiem Dat Gia ve Python 2 trong Maya 2020

#### ⚠️ Nguyen nhan 1: Xung dot cu phap Relative Import & `from __future__ import absolute_import` trong Python 2
- **Hien tuong**: Khai bao `from .core import ...` hoac `from ..core import ...` o dau file bi Maya 2020 bao `SyntaxError: invalid syntax` tren dau cham `.`.
- **Bai hoc**: Trong Python 2.7, cu phap Relative Import bang dau cham khong bao gio nen dung cho package chinh. Hay luon dung **Absolute Import** ro rang (`from Animeow_Enjo_Pipeline.core import ...` hoac `import Animeow_Enjo_Pipeline.core.file_manager as file_manager`).

#### ⚠️ Nguyen nhan 2: Bay "Loi ao" (Indirect Error Reporting) do Cu phap hong o Module con
- **Hien tuong**: Maya lien tuc bao loi `SyntaxError` o dong 10 cua `window.py` (`from Animeow_Enjo_Pipeline.core import file_manager...`), mac du cau lenh import o dong 10 hoan toan dung cu phap!
- **Bai hoc cot loi**:
  - Khi mot module con (`file_manager.py`) bi loi cu phap ben trong no (vi du: chuoi docstring `"""` bi chen de trung lap), Python 2 co gang bien dich file do khi no duoc import boi `window.py`.
  - Python 2 sau do throw ra Traceback va bao vi tri loi tai **dong `import` o `window.py`** thay vi bao thang vao file goc `file_manager.py`.
  - **Quy trinh Debug chuan**: Khi gap loi import khong ro rang tren Maya, KHONG DUOC doan mo UI ma phai dung cong cu bien dich kiem tra toan bo file:
    `python -c "import py_compile, glob; [py_compile.compile(f) for f in glob.glob('.../*.py')]"`
    Lenh nay se chi thang mat vi tri dong code bi loi thuc su!

#### ⚠️ Nguyen nhan 3: Bo nho Cache Bytecode `.pyc` cua Python 2 trong Maya RAM
- **Hien tuong**: Code Python `.py` da duoc sua tren o cung nhung Maya 2020 nap lai van bao loi cu.
- **Bai hoc**: Python 2 tu dong tao cac file compiled bytecode `.pyc` ngay canh file `.py`. Khi Maya nap module, neu file `.pyc` cu van ton tai hoac module nam trong `sys.modules`, Maya se uu tien doc bytecode loi cu.
- **Giai phap**: Phai xoa sach `sys.modules` va quet don triet de cac file `.pyc` trong thu muc truo'c khi `import`:
  ```python
  import sys, os
  for m in list(sys.modules.keys()):
      if 'Animeow_Enjo_Pipeline' in m:
          del sys.modules[m]
  for root, dirs, files in os.walk(pkg_dir):
      for f in files:
          if f.endswith(".pyc"):
              os.remove(os.path.join(root, f))
  ```
