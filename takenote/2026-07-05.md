## 1. Đa dạng hóa chế độ nạp Reference khi mở file
- Thay thế checkbox "Mở nhanh" cũ bằng dropdown **ComboBox** chứa 5 chế độ nạp Reference linh hoạt:
  1. `⚡ Mở nhanh (Không load Ref)`: Trì hoãn nạp tất cả reference để mở file nhanh nhất.
  2. `👤 Nạp Nhân vật & Camera (Characters & Cameras)`: Tự động lọc và tích chọn nạp rig nhân vật và camera rig để artist làm việc tiện lợi nhất.
  3. `📋 Tự chọn Ref (Selective Preload)`: Hiển thị bảng chọn nạp gốc của Maya.
  4. `🔝 Chỉ nạp Ref cấp 1 (Top Level)`: Chỉ nạp reference ngoài cùng.
  5. `🔄 Mở bình thường (Nạp tất cả)`: Tải đầy đủ toàn bộ reference.
- **Lưu nhớ cấu hình (Persistence):** Tích hợp `cmds.optionVar` để ghi nhớ chế độ nạp được chọn cuối cùng của artist, tự động áp dụng khi khởi động lại công cụ.

## 2. Vá lỗi Preload Reference Editor của Maya (Lỗi Zombie)
- **Hiện tượng:** Khi sử dụng Preload để mở file, Maya báo lỗi file không tìm thấy trỏ tới một file Zombie cũ (`ZIC_V2_Shot_08.0007.ma`).
- **Nguyên nhân:** Maya không tự động đồng bộ biến optionVar `preloadRefEdTopLevelFile` khi dùng lệnh `cmds.file(..., buildLoadSettings=True)` bằng code, khiến bảng Preload Editor của Maya vẫn cố nạp file cũ bị cache.
- **Giải pháp:** Tự động gán lại biến `optionVar` này bằng đường dẫn file hiện hành trước khi mở file:
  `cmds.optionVar(stringValue=("preloadRefEdTopLevelFile", normalized_path))`

## 4. Cải tiến giao diện 3 cột song song (Master-Detail-Detail)
- Tách danh sách thành 3 cột riêng biệt để tối ưu hóa quản lý:
  - **Cột 1 (🎬 Danh sách Shot):** Danh sách các Shot.
  - **Cột 2 (📂 File Maya):** Danh sách các phiên bản file Maya của Shot được chọn. Double click để mở file.
  - **Cột 3 (🎥 Playblasts):** Danh sách toàn bộ các file video Playblast nháp tương ứng của file Maya đó.
    - File video hoạt động (Active): Đánh dấu màu xanh lá `🟢 [Active]`.
    - File video phiên bản lịch sử cũ (Old): Đánh dấu màu xám `⚪ [Old]`.
- **Phát video trực tiếp:** Double click vào file video bất kỳ ở cột 3 sẽ tự động mở phát video đó bằng trình phát mặc định của máy tính (VLC, PotPlayer, v.v.) qua lệnh gọi hệ thống `os.startfile`.
- **Tối ưu hóa quét video động:** Nâng cấp hàm quét video trong thư mục nháp `mov/` để nhận diện tất cả các tệp video bắt đầu bằng tên file Maya, bao gồm cả các video có đính kèm thêm hậu tố tên camera (như `..._Lay_v02_v02_Cam_Rig_DOF_Shot_Cam.mov`).

## 6. Sửa lỗi tương thích kiểu dữ liệu trên Maya Python
- Khắc phục lỗi `'list' object has no attribute 'replace'` và `cannot concatenate 'str' and 'list'` phát sinh do hàm `cmds.selLoadSettings(..., fileName=True)` trong Maya Python trả về kết quả dưới dạng một `list` chứa 1 phần tử thay vì chuỗi.
- Vá lỗi scope biến cục bộ `UnboundLocalError: local variable 'os' referenced before assignment` trong hàm `on_open_playblast_file` và `show_file_list_context_menu` do khai báo `import os` cục bộ trong các khối điều kiện phía dưới.
