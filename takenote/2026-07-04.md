# Take Note - 2026-07-04

Hôm nay đã hoàn thành nâng cấp toàn diện pipeline quản lý file trong Maya (**Animeow Enjo Pipeline**) đáp ứng chuẩn làm việc mới của studio:

## 1. Phẳng Hoá Cấu Trúc File Làm Việc (Working Files Flat Structure)
- **Loại bỏ thư mục Shot con lẻ ở giữa**:
  - Khâu **Layout**: Các file layout block dạng `.ma` được lưu trực tiếp tại `WorkingFile\Layout\` (ví dụ: `KS_ESS_V2_Shot_01-30_Lay_v01.ma`).
  - Khâu **Animation**: Các file anim lẻ dạng `.ma` được lưu trực tiếp tại `WorkingFile\Anim\` (ví dụ: `KS_ESS_V2_Shot_01_Anim_v01.ma`).
- **Đơn giản hoá giao diện**: Loại bỏ hoàn toàn dropdown Shot, danh sách file nháp sẽ hiển thị trực tiếp theo khâu được chọn.

## 2. Nâng Cấp Chuẩn Hoá Tên Tập Phim (Folder & File Abbrev)
- **Tên thư mục trên Server (Folder Name)**: Chuẩn hoá PascalCase nối bằng dấu gạch dưới (ví dụ: `Elevator_Safety_Song_V02`, `AAA_25`).
- **Mã viết tắt của file Maya (Abbrev)**: Tự động viết tắt chữ cái đầu + version 2 chữ số (ví dụ: `KS_ESS_V02`, `LL_ESS_V02`, `KS_AAA_25`).
- **Dialog tạo tập phim mới thông minh**: Hiển thị rõ quy ước đặt tên và **Preview thời gian thực** (cả tên thư mục và mã file Maya). Cho phép Leader tự do **sửa tay (custom)** nếu không thích gợi ý mặc định.
- **Đồng bộ mạng LAN**: Cấu hình viết tắt custom của Leader được ghi vào file `metadata.json` nằm trong thư mục tập phim trên Server. Các máy Animator khác khi quét sẽ tự động đọc file này để đồng bộ 100%.

## 3. Cải Tiến Tiện Ích Giao Diện & Sửa Lỗi
- **Nút Refresh to rõ hơn**: Bổ sung nút `🔄 Làm mới` (rộng 110px, cao 26px) ở góc trên bên phải để nạp lại dữ liệu từ Server mà không làm mất lựa chọn hiện hành của artist.
- **Sửa lỗi Unicode trong Python 2**: Xử lý triệt để lỗi `UnicodeDecodeError` khi hiển thị cảnh báo tiếng Việt trong Maya 2020.
- **Sửa lỗi đổi tên trên Windows**: Thêm cơ chế giải phóng file đang mở và cảnh báo thân thiện khi thư mục/file bị lock bởi Windows Explorer.

## 4. Hệ Thống (map_drives.bat)
- Cập nhật script `map_drives.bat` chuyển ánh xạ ổ đĩa mạng phụ từ **`Y:`** sang **`V:`** (`Z:` và `V:`).
